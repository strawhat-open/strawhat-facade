# strawhat-facade

strawhat facade

## 响应码规范

### 公共响应码

公共响应码最多三位，对应HTTP STATUS 定义请参照 

```java
org.strawhat.TidingsStatus
```

### 服务响应码

服务响应码是各应用根据自身业务特性定义的响应码，模板如下

```java
   1000         00             000
 |系统编号|   |子系统编号|   |响应码区分编号|
```
